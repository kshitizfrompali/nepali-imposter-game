Game Screen Flow

5 screens — all client-side, no backend

Screen 1

🏠 Home

Game title · Language toggle (EN / नेपाली) · "Start Game" button

↓
Screen 2

⚙️ Settings

Number of players (2–10) · Number of imposters (1 to players−1) · Word mode selector

If "Pick category" mode: show category list

↓
Screen 3

👁️ Player Reveal (pass-and-play)

"Pass to Player 1 — tap to see your role" → tap → shows word+category OR "IMPOSTER" → "Done, pass on"

Repeats for each player. Progress: "3 / 6 players"

↓
Screen 4

💬 Discussion (live)

All players have seen roles — "Discuss now!" message · Big "Reveal Imposters" button

↓
Screen 5

🎭 Reveal

Shows imposter names · The secret Nepali word · "Play Again" button

All state lives in React — no server, no database
