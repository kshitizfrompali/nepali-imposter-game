Tech Stack Approaches

Pick the foundation for your Nepali Imposter Game

A

Vanilla HTML/CSS/JS

No framework, no build step. Pure browser code.

Pros

  • Zero setup
  • Works everywhere
  • Tiny file size

Cons

  • Hard to manage multi-screen state
  • No component reuse
  • GitHub Pages deploy is manual
B ⭐ Recommended

React + Vite → GitHub Pages

Component-based UI, fast dev server, one-command deploy.

Pros

  • Clean multi-screen state (settings → reveal → result)
  • Easy GitHub Pages deploy via gh-pages
  • Wide ecosystem & documentation

Cons

  • Slightly heavier bundle
  • Requires Node.js locally
C

Svelte + Vite → GitHub Pages

Compile-time framework — smaller output, very clean syntax.

Pros

  • Smallest bundle size
  • Simpler code than React
  • Great for small apps like games

Cons

  • Smaller community
  • Less familiar to most devs
